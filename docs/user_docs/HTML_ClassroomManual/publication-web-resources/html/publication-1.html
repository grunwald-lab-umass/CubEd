A Class 3R laser is low powered, which normally does not harm eyes during momen-tary exposure. This duration is within the aversion response, when a person turns away and/or blinks to avoid bright light.

Do not deliberately look or stare into the laser beam. Laser protective eyewear is normally unnecessary. A Class 3R laser is neither a skin nor materials burn hazard.

However, a Class 3R laser can be a distraction, glare or flash blindness hazard for pilots and drivers. NEVER aim any laser towards an aircraft or a vehicle in motion. This is unsafe and is illegal – you could be arrested and jailed.

ONLY ALLOW USE BY RESPONSIBLE PERSONS!!!

This is NOT a toy. Continuous adult supervision is required for children to safely use Class 3R lasers.
