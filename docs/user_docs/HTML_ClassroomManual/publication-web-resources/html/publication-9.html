Mirror Alignment

Mirrors can be used to direct the beam. The CubeEd mirrors are orientated within the cube allow a 90° redirection, give or take with adjustments; i.e. the mirrror cubes enable the beam to travel around corners.

Horizontal mirrors can be used to direct the beam path 90° left or right. Vertical mir-rors can be used to direct the beam path 90° up or down.

Adjust the mirror angle with the set screws to keep the beam path on the same plane.

Tip: Use the same paper method as that used in the beam collimation step 3 to ensure that the beam path stays straight.
