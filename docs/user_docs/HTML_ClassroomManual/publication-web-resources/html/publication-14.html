Joystick – Controls the movement of the stage. Does not contribute to speed control. Push the joystick all the way in one direction at a time to move the stage. Movement of the x- and y-axes are controled in one mode, the z-axis is controlled in a second mode.

Start – Switches between the two joystick modes: x/y and z (z-mode only uses the vertical axis of the joystick). Hold for one second.

Select – Resets the stage to the “home” position: hold for one second.
