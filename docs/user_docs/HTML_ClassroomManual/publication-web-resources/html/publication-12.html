Emission Beam Path

Let the beam pass through the filter cube and onto a vertical mirror. Adjust the mirror angle via its set screws to maintain a parallel beam path, directing the path forward.

Place the camera so the camera sensor is about 200 mm from the f200 lens.

Place a f200 lens in front of the beam. Center the lens using the set screws.

Confirm the alignment of the beam path as before.

Note: The photos in this section were taken of a fully constructed CubEd. Add each optical component sequentially, checking the beam path as each element is added.

Tip: Use the same alignment procedure as that used in the beam collimation.
