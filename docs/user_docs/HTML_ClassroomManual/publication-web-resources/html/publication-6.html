Laser Alignment

Place a laser source (green1) on top of two base cubes (black), and position it on the magnetic base plate. Remove the covers from the laser source and unscrew the four set screws enough to loosen them a little bit.

Place two targets in front of the laser (see image below), at least three cube lengths away from each other and one cube length away from the laser.

Set up a flat target such as paper on the other end of the targets, to see how the laser needs to be adjusted.
